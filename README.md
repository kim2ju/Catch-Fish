# 생선집 막내백수

프로젝트 명: 생선집 막내백수

과목: 인터렉티브 미디어 디자인

목표: 아두이노와 유니티를 이용하여 유저의 움직임에 따라 반응하는 게임 개발

기간: 2023.04.04 ~ 2023.06.07

참여 인원(기여도): 4명(25%)

역할: 기획, 프로그래밍

수행 업무: 게임 컨셉 등 전반적인 기획, 아두이노 회로 설계 및 코딩, 게임 로직 설계, 플레이어 이동/블루투스 통신/게임 종료 등 구현
